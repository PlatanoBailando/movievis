Character Network {{activeSheetName}}

The graph below is a representation of character interactions within the narrative. Each character represents a node (circle) where the size of the node is dependent on the number of scenes that character appears in. If two characters appear in a scene together, they are linked by a blue line. This analysis provides insight into character narrative importance, as measured by their centrality to the story (node size) and connectedness to other characters (links). The filter allows you to add/remove characters from the graph based on gender and racial group membership.
